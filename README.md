# Clone-GitHub

## Descrição
Este projeto é uma recriação da página de login do GitHub. O objetivo é praticar HTML, CSS e JavaScript.

## Tecnologias Utilizadas
- HTML
- CSS
- JavaScript

## Como Usar
1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.
